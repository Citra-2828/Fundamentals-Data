# 📊 Proyek Analisis Data E-Commerce

## Deskripsi
Proyek ini menganalisis data e-commerce untuk menemukan pola penjualan dan perilaku pelanggan.

## Cara Menjalankan

pip install -r requirements.txt  
streamlit run dashboard/dashboard.py  

## Insight
- Puncak penjualan terjadi saat Black Friday
- Kategori Bed, Bath & Table memiliki revenue tertinggi
- Banyak pelanggan termasuk Lost Customers
